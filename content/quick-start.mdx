# 安装与快速开始

## 1. 下载桌面版

前往 [下载中心](/download)，官网会尝试自动推荐与你当前设备最匹配的安装包。

如果你需要手动选择：

- `macOS Apple Silicon` 优先下载 `arm64`
- `Windows` 默认下载 `x64`
- `Linux` 根据你的发行版与架构选择对应产物

## 2. 启动 PRDKit Desktop

安装完成后，打开桌面应用，你会先进入工作台入口页。这里通常会聚合几个最常用动作：

- 打开最近项目
- 创建新的 PRD 或原型
- 进入当前项目进行编辑、标注和版本管理

## 3. 建议的第一轮体验

最推荐的新手路径不是零散点击，而是一口气完成一个闭环：

1. 创建一个新需求主题
2. 生成 PRD 初稿
3. 做一个对应原型页面
4. 对关键位置补上标注
5. 形成一个 checkpoint 版本

## 4. 如果你更偏 CLI 工作流

PRDKit 并不是只能通过桌面端使用。很多团队会把 CLI 作为自动化和脚手架入口，再把 Desktop 作为高频协作界面。

你可以继续阅读：

- [工作流概览](/docs/workflow)
- [CLI 与 Desktop](/docs/desktop)
