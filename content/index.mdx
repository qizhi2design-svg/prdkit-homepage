# PRDKit 文档总览

PRDKit 官网首版文档聚焦在几个最常用的入口，帮助你快速理解这套工具是如何围绕产品经理工作流组织起来的。

## 你可以从这里开始

- 如果你是第一次接触 PRDKit，先看 [安装与快速开始](/docs/quick-start)
- 如果你想知道整条链路是怎么串起来的，先看 [工作流概览](/docs/workflow)
- 如果你更关心工具边界和桌面端定位，先看 [CLI 与 Desktop](/docs/desktop)
- 如果你要把成果分享给团队，继续看 [发布与分享](/docs/sharing)
- 如果你希望系统地浏览技能与使用场景，阅读 [产品使用指南](/docs/product-guide)

## 官网首批文档策略

这批文档不是把仓库里的 `docs/` 全量搬过来，而是做了一次筛选：

- 保留高频、对外可读的内容
- 优先覆盖下载、上手、工作流和产品定位
- 暂不引入博客、复杂 CMS 或大规模历史设计文档

## 你会在文档里看到什么

- PRDKit 的核心能力如何围绕产出物组织
- Desktop 为什么是第一优先级入口
- 文档、原型、标注、版本和分享是如何串起来的
- 下载与版本分发的当前约定
