# 文档总览

PRDKit 是产品经理的 AI 工作台——从方案构思、写 PRD、做原型到发布分享，集中在一个桌面应用里完成。

---

## 按你的角色来读

**第一次接触，想知道怎么上手**
→ [快速开始](/docs/quick-start) —— 下载、安装、十分钟走完第一轮闭环

**想搞清楚界面上每块区域是干什么的**
→ [工作台漫游](/docs/workbench) —— 文件树、画布、AI 面板、三种模式

**准备正经写一份 PRD**
→ [撰写 PRD](/docs/prd-lifecycle) —— 编写、AI 辅助、版本管理和历史对比

**在做原型，需要加标注评审**
→ [原型与标注](/docs/prototype) —— 画布操作、元素选择、标注创建

**做好了，想发出去给团队看**
→ [发布与分享](/docs/sharing) —— 发布版本、分享链接、密码保护

**习惯命令行，需要批量和自动化**
→ [CLI 参考](/docs/cli-ref) —— init / clone / prd / prototype 命令大全
